Veb-razrabotka

Всего существует несколько атрибутов веб-разработки:

  • HTML
  • CSS
  • JS


HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — язык разметки,
который используется для создания и структурирования веб-страниц. Он помогает определять,
как содержимое страницы должно отображаться в браузерах.

HTML

CSS (Cascading Style Sheets, «каскадные таблицы стилей») — язык стилей, с помощью которого
оформляют веб-страницы. Он определяет, как должны отображаться элементы HTML, включая
макет, цвета, шрифты и другие визуальные аспекты.

CSS

JavaScript (JS) — язык программирования, который изначально был создан для добавления интерактивности на веб-страницы.
Поддерживает такие стили:

  1. объектно-ориентированный;
  2. императивный;
  3. функциональный.

JavaScript обычно используется как встраиваемый язык для программного доступа к объектам приложений.
